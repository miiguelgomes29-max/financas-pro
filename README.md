# 💰 Finanças Pro - Controlo Financeiro Pessoal

Este é um projeto desenvolvido para a disciplina de **PSI** (Programação e Sistemas de Informação). Consiste num Dashboard web dinâmico para gestão e controlo financeiro pessoal centralizado.

---

## 🚀 Funcionalidades Principal

* **Gestão de Saldos:** Cálculo automático do saldo atual com base em receitas e despesas.
* **Histórico de Atividades:** Listagem em tempo real dos movimentos recentes.
* **Gráficos Estatísticos:** Visualização em linha do fluxo de dinheiro ao longo das semanas.
* **Sistema de Rotas Protegido:** Controlo de acessos estruturado e seguro contra Directory Traversal.

---

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **Base de Dados:** MySQL (através do phpMyAdmin)

---

## 💻 Como Executar o Projeto Localmente

Como o projeto utiliza PHP e MySQL, necessita de um ambiente de servidor local para correr:

1. Instalar o **XAMPP** (ou similar).
2. Colocar a pasta do projeto dentro do diretório `htdocs`.
3. Iniciar os módulos **Apache** e **MySQL** no painel do XAMPP.
4. Importar o ficheiro da base de dados (`.sql`) no **phpMyAdmin**.
5. Aceder ao navegador através de: `http://localhost/nome-da-tua-pasta/index.php`.
